void pg_firewall_gc ( struct pg_brick *  brick)

Manually call the firewall garbage collector. NPF firewall tracks all connexions when stateful. To clean old connexions, we must manually call the connexion garbage collection when PG_NO_CONN_WORKER flag shas not been set at brick creation.

Add a new rule in the firewall. Note that the rule won't be effective, you will need to call firewall_reload() before. Offcourse, you can call firewall_rule_add several times before firewall_reload().

Parameters
brickpointer to the firewall brick
filtera pcap-filter string to apply, if a NULL string is passed, it will allow all traffic.
sideside where the filter will occurs, you can use SIDE_MAX to specify both sides.
statefulallow packet inspection to distinguish legitimate packets for different types of connections.
errpis set in case of an error
Returns
0 if the rule has been correctly built and added -1 on error and errp is set.
