◆ pg_rxtx_tx_callback_t

typedef void(* pg_rxtx_tx_callback_t) (struct pg_brick *brick, pg_packet_t **tx_burst, uint16_t *tx_burst_len, void *private_data)

The tx callback is called each time the brick is polled.

Parameters
brickpointer to the rxtx brick
tx_burstarray of pre-allocated packets. The array size if given with tx_burst_len. Use pg_packet_data to get a pointer where to write data. Use pg_packet_set_len to set len of your packet. You must not write data once callback is done. You will get back the same packet array at each call. You can take advantage of this to only write what you need to.
rx_burst_lennumber of packets ready to be sent. Must not be > PG_RXTX_MAX_TX_BURST_LEN
private_datagive back user's data
